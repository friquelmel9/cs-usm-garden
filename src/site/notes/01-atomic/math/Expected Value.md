# Expected Value
In [[Probability Theory\|Probability Theory]], the Expected Value (expectation, mean, or first [[Moment\|Moment]]) is a generalization of the weighted average.

The expected value of a [[Random Variable\|Random Variable]] with a finite number of outcomes is a weighted average of all possible outcomes.

> [!formula] Expected Value for Random Variable $X$ with finitely many outcomes
> Consider a Random Variable with a finite list of values $x_1, x_2, \dots, x_n$ with [[Probability\|Probability]] $p_1, p_2, \dots, p_n$ respectively. The expectation is defined as
> #formula $$E[X] = x_1p_1 + x_2p_2 + \cdots + x_np_n$$

> [!formula] Expected Value for Random Variable with Density
> If $X$ is a [[Random Variable\|Random Variable]] with a [[Probability Density Function\|Probability Density Function]] given by $f$, the result is given by the [[Integral\|Integral]] on the interval
> #formula $$E[X] = \int_{-\infty}^\infty x f(x) dx$$
