# Object-Oriented Programming

Object-Oriented Programming (OOP) is a [[Programming Paradigm\|Programming Paradigm]] in [[Computer Science\|Computer Science]] that relies on the concept of [[01-atomic/computer-science/Class\|Classes]] and [[01-atomic/computer-science/Object\|Objects]]. It is used to structure a software program into simple, reusable pieces of code which are used to create individual instances of objects.

## Building Blocks in OOP
- [[01-atomic/computer-science/Class\|Class]]: Blueprint for structures with [[01-atomic/computer-science/Method\|Method]] and [[Attribute\|Attribute]]
- [[01-atomic/computer-science/Object\|Object]]: Instance of a class
- [[Attribute\|Attribute]]: Information stored in a class
- [[01-atomic/computer-science/Method\|Method]]: Function for the class
## Principles of OOP
- [[Inheritance\|Inheritance]]
- [[Encapsulation\|Encapsulation]]
	- [[Access Identifiers\|Access Identifiers]]
- [[Polymorphism\|Polymorphism]]
	- [[01-atomic/computer-science/Function Overloading\|Function Overloading]]

## Languages
There are languages that support OOP. Some of them are:
- [[C++\|C++]]
- [[Python\|Python]]

There are languages that work with OOP exclusively. Some of the are:
- [[C-sharp\|C-sharp]] (C#)
