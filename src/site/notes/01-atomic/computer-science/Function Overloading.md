# Function Overloading
Function Overloading is the ability to create multiple [[Function\|Functions]] of the same name, that differs in implementation depending on the context of the call.

Common forms of overloading are:
- Using [[Parameter\|Parameter]]: `myFunction() != myFunction(int x)`
- Using different [[Data Type\|Data Type]]: `myFunction(int x) != myFunction(float x)`

Not all [[Programming Language\|Programming Languages]] support overload. Some are:
- [[C++\|C++]]
