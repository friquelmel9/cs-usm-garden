# Class

In [[01-atomic/computer-science/Object-Oriented Programming\|Object-Oriented Programming]], a class is a syntactic entity structure used to create [[01-atomic/computer-science/Object\|Objects]]. The capabilities of a class depend on the [[Programming Language\|Programming Language]], but the shared aspects are the [[Attribute\|Attributes]] and [[01-atomic/computer-science/Method\|Methods]] that can be associated with a particular object or all objects in a class.

If the language supports [[Inheritance\|Inheritance]], a class can be defined based on another class with all of its state and behavior plus additional state and behavior that further specializes the class.
