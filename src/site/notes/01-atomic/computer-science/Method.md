# Method
A method is a function in [[01-atomic/computer-science/Object-Oriented Programming\|Object-Oriented Programming]] associated with an [[01-atomic/computer-science/Class\|Class]] that takes and modifies the state of the [[01-atomic/computer-science/Object\|Object]].

## Implementations
Methods also could support|
- [[Method Override\|Method Override]]: When a method on a [[SuperClass\|SuperClass]] is override by the [[SubClass\|SubClass]] ([[Inheritance\|Inheritance]])
- [[01-atomic/computer-science/Function Overloading\|Function Overloading]]: When a method changes based on the [[Parameter\|Parameters]] given 

## Types of Methods
- [[01-atomic/computer-science/Constructor\|Constructor]]: An method that runs when an object is created
- [[01-atomic/computer-science/Destructor\|Destructor]]: An method that runs when the object is destroyed
- [[Static Method\|Static Method]]: Methods relevant to all instances of a class
- [[Abstract Method\|Abstract Method]]: Used on [[Interface\|Interfaces]] to declare methods
