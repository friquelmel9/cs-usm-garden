# Destructor
In [[01-atomic/computer-science/Object-Oriented Programming\|Object-Oriented Programming]], a destructor is a special type of [[01-atomic/computer-science/Method\|Method]] for a [[01-atomic/computer-science/Class\|Class]] called to destroy an [[01-atomic/computer-science/Object\|Object]], generally before the memory assigned to he object is deleted. This can happen when the lifetime of the object is bound to a [[Scope\|Scope]] and the execution leaves the scope. 

The main purpose of a destructor is to free [[Resources (Computer Science)\|resources]] which were acquired by the object during it's lifetime
