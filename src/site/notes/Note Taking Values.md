# Note Taking Values

## Sobre las notas
Me tuve que crear una nueva vault para transformar las notas para seguir un nuevo principio. Me di cuenta que en cierto punto no escribía las notas yo, si no que copiaba y pegaba (incluso aveces de la propia IA).  

Ahora, el objetivo es que las notas atómicas se encuentren escritas de tal modo que sepa que es cada concepto. No debo sabérmelas de memoria, pero entender que estoy viendo. Por eso mismo, generalmente estas notas quizás no enseñan de la mejor manera, están escritas primero para mi.

## Sobre los contenidos

Hay cinco tipos de contenidos dentro de la vault.
- **atomic**: Las notas atomicas
- **moc**: map of contents, contienen agrupamiento de notas según algún tópico en particular
- **resource**: Notas que hacen referencia a recursos (tales como libros, PDF, o sitios web). Si por algún necesario se deba copiar contenido de internet, quedara guardado de la forma #clippings 
- **templates**: self
- **media**: Imagenes, generalmente en `.svg` para guardar espacio

Lo importante es moverse mediante los **moc**, siendo el principal de este repositorio el [[02-moc/Universidad\|Universidad]].
